# 动态表格生成器（Dynamic Table Generator）

一个纯前端的 9:16 动态榜单制作工具。  
通过 JSON 数据输入，可快速生成可视化表格、逐行动画，并导出短视频。

## 项目功能分析

### 1) 内容配置能力
- 标题支持文本、字号（12-60）、颜色实时调整。
- 表头字号支持手动覆盖（0=自动，>0 使用自定义值）。
- 提供 10 套内置背景主题，可一键切换预览风格。

### 2) 数据处理能力
- 数据来源为 JSON 数组，输入变化后会自动重绘表格。
- 自动提取第一行字段作为列头，`排名变动` 字段不作为独立列展示。
- 若 `排名变动` 为“上升/下降”，会在“排名”单元格中追加 ↑ / ↓ 指示。

### 3) 表格自适应布局能力
- 根据数据行数动态计算字体、行高、内边距，保证 9:16 画布内可读性。
- 自动计算最后一行剩余高度，避免出现底部留白或截断。
- 对“排名”“财富”“公司”等列做差异化样式增强，提升信息层级。

### 4) 动画播放能力
- 支持 5 种逐行入场动画：
  - 向下展开（expand）
  - 左侧飞入（slide-left）
  - 右侧飞入（slide-right）
  - 下方浮入（rise-up）
  - 缩放淡入（zoom-in）
- 支持间隔可调（50-3000ms，步进 50ms）。
- 播放时会重置行状态，按顺序逐条展开，避免动画残留。

### 5) 视频导出能力
- 点击“导出视频”后会自动执行：重渲染 → 启动录制 → 播放动画 → 停止并下载。
- 录制对象是右侧预览区（9:16），导出内容与预览一致。
- 基于 `Canvas.captureStream + MediaRecorder` 实现，优先 MP4，不支持时自动回退 WebM。
- 录制与预览均可在浏览器本地运行，无需后端服务。

## 使用方式

### 1. 快速开始
1. 直接用浏览器打开 `index.html`。
2. 在左侧输入或粘贴 JSON 数据。
3. 调整标题、背景、动画参数。
4. 点击“动画预览”或“导出视频”。

### 2. 数据格式示例
```json
[
  {
    "排名": 1,
    "姓名": "张三",
    "财富(亿元人民币)": 2500,
    "主要公司": "科技集团",
    "排名变动": "上升"
  },
  {
    "排名": 2,
    "姓名": "李四",
    "财富(亿元人民币)": 2200,
    "主要公司": "投资公司",
    "排名变动": "下降"
  }
]
```

## 技术实现

- `HTML5`：配置面板与预览区结构。
- `CSS3`：深色主题、玻璃感容器、逐行入场动画。
- `JavaScript (ES6+)`：配置联动、JSON 解析、表格渲染、节流式重绘。
- `Canvas 2D`：逐帧绘制预览内容，保证录制画面稳定。
- `MediaRecorder API`：编码视频流并自动下载文件。
