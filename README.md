# 动态表格生成器（Dynamic Table Generator）

一个纯前端的 9:16 动态榜单制作工具。通过 JSON 数据输入，可快速生成可视化表格、逐行动画，并导出短视频。

## 功能概览

### 1. 标题配置
- 标题文本内容实时编辑
- 标题字体大小调节（12-60px）
- 标题字体颜色选择
- 表头字体大小手动覆盖（0=自动计算）

### 2. 背景配置
- 10 套内置渐变背景主题一键切换
- 支持自定义背景图片上传
- 背景图片透明度调节（0-100%）
- 背景图片与主题叠加效果

### 3. 数据输入
- JSON 数组格式数据输入
- 数据变化后自动重绘表格
- 自动提取字段作为表格列
- 支持排名变动显示（上升↑/下降↓）

### 4. 列配置
- 列宽权重分配（自动归一化）
- 单元格字体大小独立设置
- 单元格字体颜色独立设置
- 单元格字体加粗开关
- 表头字体大小独立设置
- 表头字体颜色独立设置

### 5. 动画系统
- 5 种逐行入场动画：
  - 向下展开（expand）
  - 左侧飞入（slide-left）
  - 右侧飞入（slide-right）
  - 下方浮入（rise-up）
  - 缩放淡入（zoom-in）
- 动画间隔可调（50-3000ms，步进 50ms）
- 动画预览功能

### 6. 弹幕系统
- 弹幕开关控制
- 弹幕内容批量输入（每行一条）
- 弹幕字体大小调节（10-48px）
- 弹幕字体颜色选择
- 弹幕随机垂直位置布局
- 循环滚动动画

### 7. 自适应布局
- 根据数据行数自动计算字体大小
- 根据数据行数自动计算行高
- 自动调整最后一行高度
- 确保 9:16 画布内完全显示
- 差异化样式（排名、财富、公司列增强）

### 8. 视频导出
- Canvas 逐帧渲染保证画面稳定
- MediaRecorder API 录制
- 优先导出 MP4 格式，不支持时回退 WebM
- 自动下载视频文件

## 使用方式

### 1. 快速开始
1. 直接用浏览器打开 `index.html`
2. 在左侧输入或粘贴 JSON 数据
3. 调整标题、背景、动画参数
4. 点击"动画预览"或"导出视频"

### 2. 数据格式示例
```json
[
  {
    "排名": 1,
    "姓名": "张三",
    "财富(亿元人民币)": 2500,
    "主要公司": "科技集团",
    "排名变动": "上升"
  },
  {
    "排名": 2,
    "姓名": "李四",
    "财富(亿元人民币)": 2200,
    "主要公司": "投资公司",
    "排名变动": "下降"
  }
]
```

## 技术实现

- **HTML5**：配置面板与预览区结构
- **CSS3**：深色主题、玻璃感容器、逐行入场动画
- **JavaScript (ES6+)**：配置联动、JSON 解析、表格渲染、节流式重绘
- **Canvas 2D**：逐帧绘制预览内容，保证录制画面稳定
- **MediaRecorder API**：编码视频流并自动下载文件
